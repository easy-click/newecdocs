## TOC

Rainbond 社区 TOC (technical oversight committee) 成员的主要职责：

* 建立项目总体技术方向和设计体系
* 参与并评审 Proposal
* 审核并处理 Github 的 PR 和 issue
* 解决贡献者提出的技术问题
* 建立特殊兴趣小组(SIG)，吸引更多的贡献者
