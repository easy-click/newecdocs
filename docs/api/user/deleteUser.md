## 删除用户

根据用户id删除用户

```json title="请求路径"
DELETE /openapi/v1/users/{user_id}
```

## 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|user_id|path|string| 是 |none|


## 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|204|[No Content](https://tools.ietf.org/html/rfc7231#section-6.3.5)|成功|Inline|
